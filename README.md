# awscli
A docker image with awscli used for test for something else.


## QuickStart

```sh
docker run -d \
    -e AWS_ACCESS_KEY=xxxxx \
    -e AWS_ACCESS_KEY=xxxxx \
    icmdb/awscli 
```
